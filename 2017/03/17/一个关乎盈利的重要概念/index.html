一个关乎盈利的重要概念

发表于 2017-03-17 | 更新于 2017-03-19 | 分类于 交易 ， 市场分析 | |
判断对错不重要，重要的在于正确时获取了多大利润，错误时亏损了多少！

一、​故事

　　在斯坦利·杜肯米勒（Stanley Drucken-miller）为索罗斯旗下的量子基金开始工作不久，由于准确判断了美元的弱势前景，他大量沽空美元买入马克。市场的发展对他越来越有利，杜肯米勒也感到很骄傲，并来到索罗斯的办公室谈起这笔交易。
　　“你建了多大的仓位？”索罗斯问杜肯米勒。
　　“10亿美元。”
　　“你把这也叫仓位？”索罗斯不屑地反问。这是华尔街的一则经典问答。在金融大鳄索罗斯的耳提面命之下，斯坦利·杜肯米勒学到的最有价值的投资心得，并不是弄清楚投资的对错，而是清楚自己在投资正确的时候是不是能够乘胜追击，投资失败的时候是不是能及时止亏。在杜肯米勒刚刚加盟索罗斯旗下的量子基金的时候，索罗斯对他最大的批评，就是尽管对于市场大势的判断准确，但没有能最大限度地充分利用机会赚取最大利润。

二、概念

　　从索罗斯的名言来看：在投资市场的成功的秘诀或秘籍只有一个。那就是确保赢的时候赢的多，输的时候输得少。所以，我们必须从一个系统化的角度来看待投资或交易。

而上面这个表述，如果用公式来表示，我觉得可以写成这样：

盈利概率=方向判断的准确率*报酬风险比(报风比)

交易最终要想取得一个正向的结果，我们必须努力提高这个盈利概率。

这里面的报风比是反映投资交易中要获得可能受益所冒的风险。通常认为每笔交易的报风比应该不小于3，即：想获得3块钱的报酬，可能要付出1块钱的止损，或者说冒一块钱风险的一笔交易最终获利了3块的报酬。长期来看，投资盈亏比，才是直接反映投资者综合水平的的一个量化指标。

如果你在每一笔的交易之前都能坚持这样量化准备，争取每一笔交易的报风比都大于3，那么你会发现：就算你的正确率只有35%， 长期而言，你最终的交易结果将会是一个正向的预期值。

三、实例

以下是今天盘中写的一些内容：
《一笔可以值得考虑的交易》

我们先抛开9月鸡蛋的基本面和技术分析（比如跌破上涨趋势线），只从报风比的角度来分析一笔做空的交易：
假设今天收盘价格是目前的日内均价4234，如果选择进场做空，则第一目标位设在黄色线4085位置，而相应的空单止损放在今日高点4253处。

那么这笔交易可能的报酬和风险比率为：
（4234-4085）/（4253-4234）
结果超过7！

单从报风比来说，这是一笔可以值得考虑的交易！
