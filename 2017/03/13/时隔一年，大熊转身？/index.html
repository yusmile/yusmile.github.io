时隔一年，大熊转身？

发表于 2017-03-13 | 更新于 2017-03-19 | 分类于 交易 ， 市场分析 | |
目前到底是牛市还是熊市呢？

一、实体经济明显好转？

1、预期

去年一年大宗商品价格和部分地区房价的持续上涨，让大部分交易者可能都产生了一种思维惯性——价格高的会继续走高。不少人可能都忘记了本轮大宗的上涨是始于2008年下跌之后的，就是说此次上涨到底是长期下跌趋势中的一个中期反弹走势还是一个长期上涨趋势开始的第一波，这是需要每个交易者需要思考和定性的事情。

今年开年之后，似乎很多交易者会感觉实体经济明显好转，特别是PPI指数、PMI指数、CPI价格、GDP增速等指标都不断创出两年来的新高；网络上的不少财经大V都认为：今年将是货币再次大放水严重超发的一年，所有的资产价格包括商品和股市，都会因为货币的增发而将出现严重的通胀；甚至有个别还大声呼喊：中国经济已成功筑底了，今年中国经济将再起辉煌，应该要抓紧时机及时买入……

2、现实

在春节之后，上述现象真的有逐渐增强的趋势，不少商品也再次发动上攻行情。

但，现实确实如此吗？

今年年后，笔者陆陆续续和几个各地不同情况的朋友小聊了下，似乎大家都感觉自己身边的实体经济其实还不如从前了。上周又听一朋友提及最近厦门的再就业市场和往年年后情况有所不同：不再延续以前那红红火火的热闹景象，似乎很多企业都没了招人的意愿，而往年这个时候正是一些企业招兵买马准备大干一番的时间节点啊。这些情况能说明什么问题吗？

二、直观期市

个人认为：如果认定技术分析作为主要交易手段的交易者，那么他主要还是应该依据市场给出的信号，并且一定要养成习惯：根据图表给出的信号，而非听到的消息来作出交易决策。

1、3个指数

从月线的长期图表来看：国内外两个大宗商品的月线走势是略有不同的，明显国内强于国外：CRB指数目前还处在2008年那波下跌的低点之下。
这里面可能有两个原因：这两个指数编制的方法不太相同，国内外大宗商品的涨幅也不太一样。


波罗的海干散货指数是由几条主要航线的即期运费加权计算而成,为即期市场的行情的反映。认真观察，发现这个日线走势图还挺有意思的，目前这个阶段有两大可能：价格在前高附近遇阻，形成双顶；价格突破前高，继续上涨趋势。

2、股市


笔者已经连续两周对上证指数的短期上涨存在疑惑，现在仍然保持这个观点。如果选择做多个股的朋友，请注意后期大盘是否能及时返回3230之上（大概就是图中5日和20日均线位置）。

当大盘不好时，个股走好是有可能的，但是属于低于平均概率水准的。

是否交易，关键在于你进出场选择的交易周期！

就目前而言，个人认为选择短周期的朋友（就是通常意义上的短线交易）可以考虑先在场外观察情况，等大盘走好再进场，胜率更高

三、应对策略

成功的交易者也预测市场走势，但他从不与市场行情本身争论，他只是客观地根据市场的现实状况作出反应，不会设法去寻找这样或那样的主观理由来证明自己的正确。

世界上没有两片完全相同的叶子。 ——德国哲学家莱布尼茨

本系列的文章均是个人一些在实盘交易中的看法，是指导本人近期交易的一个大概方向的判断。

在实盘交易时，因为每一个交易者交易系统的独特性、个性的不同以及对应交易周期的选择不同，所以大家进出场之前，还是应当要按照自己的交易体系，在个人交易周期内确认交易目标的趋势是否发生变动，并做好相应对策——在牛市，逢低做多；在熊市，逢高放空。

对待目前这行情，如果你的交易系统继续看多，那么就利用此次下跌，逢低建仓吧；如果你的交易系统以及转为熊市，那么后面就是等待时机，逢高放空了。

祝大家：交易顺利！盈多亏少！ ^_^
